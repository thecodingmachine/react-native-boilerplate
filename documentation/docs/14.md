# Audits

Stable Jack has been audited several times.\
\
The latest audit report by Dedaub, dated December 24, 2024 can be reached on this [link](https://dedaub.com/audits/stable-jack/stable-jack-dec-24-2024/).

Moreover, Stable Jack works with Hexagate, a threat tool that scans the contracts and pauses them when an attack is detected, allowing us to continuously secure the protocol.
